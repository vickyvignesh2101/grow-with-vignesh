Student Profile
Management System

The Student Profile Management System is a responsive web application developed to simplify student record management. It allows administrators to add, update, delete and search student information through an intuitive dashboard. The project focuses on creating a clean user interface, responsive layouts and efficient data organization while providing a better user experience.

Key Features

👨‍🎓 Student Registration
✏ Update Records
🗑 Delete Student
🔍 Search Function
📱 Responsive UI
⚡ Fast Performance
